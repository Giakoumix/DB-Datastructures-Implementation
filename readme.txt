Για την δημιουργία του εκτελέσιμου το οποίο δείχνει
τις δυνατότητες της βιβλιοθήκης BF, τρέξτε την εντολή:

make bf;

Για να τρέξετε το εκτελέσιμο:

./build/bf_main

Αντίστοιχα και για τα άλλα εκτελέσιμα.
make ht;
make hp;

Σε καθε main που περιλαμβανει κατακερματισμο τρεχω την HashStatistics οπου επιστρεφει τα στατιστικα στοιχεια.

Καθε δομη αρχειου εχει ενα δευτερευον αρχειου με utilities αναλογα με την δομη
(hp_file.c: utils.c) (ht_table.c: ht_utils.c) (sht_table.c: sht_utils.c).

Στα αρχεια κατακερματισμου για να αποφυγω το συνεχες γραψιμο του block 0 στον δισκο που ειναι μια κοστοβορα 
εργασια, κατα την δημιουργια του ht/sht_info το κανω dirty και στο κλεισιμο του αρχειου μονο το κανω unpin για 
να μπορει να γραφτει στο δισκο.
